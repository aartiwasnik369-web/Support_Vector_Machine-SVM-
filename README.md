# Support Vector Machines (SVM)

## Objective
Use Support Vector Machines (SVM) for linear and non-linear classification.

---

## Tools Used
- Scikit-learn
- NumPy
- Matplotlib
- Pandas

---

## Dataset
Breast Cancer Dataset  
Binary Classification:
- Malignant (M)
- Benign (B)

---

## Steps Performed

1. Loaded and prepared the dataset for binary classification.
2. Applied train-test split.
3. Performed feature scaling using StandardScaler.
4. Trained SVM with:
   - Linear Kernel
   - RBF Kernel
5. Evaluated model performance using accuracy.
6. Tuned hyperparameters:
   - C
   - gamma
   using GridSearchCV with cross-validation.
7. Visualized decision boundary using 2 selected features (2D visualization).

---

## Concepts Covered
- Margin Maximization
- Support Vectors
- Kernel Trick
- Hyperparameter Tuning
- Cross-Validation
